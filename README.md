# 🏛️ MindPalace

A first-person, Wolfenstein-style memory palace for exploring GitHub repositories.

## Live URL

https://chrisalunlloyd2-sudo.github.io/mind-palace/

## Source

The deployed site lives in `docs/` and is auto-built from `scripts/`.

## Controls

- **WASD / arrows** — walk
- **Mouse** — look
- **Click** — lock pointer
- **P** — archivist login

Default archivist: `viper:clamchowder` (client-side gate only)
